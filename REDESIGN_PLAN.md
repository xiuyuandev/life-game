## 1. и®ҫи®Ўе“ІеӯҰ

### ж ёеҝғж„ҝжҷҜ

**и®©зҺ©е®¶з”ЁжёёжҲҸзҡ„ж„ҹи§үжқҘжҺҢжҺ§дәәз”ҹгҖӮ**

дёҚжҳҜ"з»ҷеҫ…еҠһдәӢйЎ№еҲ—иЎЁеҘ—дёҖеұӮ RPG зҡ®иӮӨ"пјҢиҖҢжҳҜ**зңҹе®һз”ҹжҙ»жң¬иә«е°ұжҳҜжёёжҲҸ**гҖӮзҺ©е®¶жҠ•е…Ҙзҡ„жҜҸдёҖеҲҶй’ҹзңҹе®һж—¶й—ҙйғҪеңЁжёёжҲҸдёӯжңүеҜ№еә”зҡ„еҸҚйҰҲгҖӮжёёжҲҸдёҚжҳҜз”ҹжҙ»зҡ„еҪұеӯҗвҖ”вҖ”жёёжҲҸе°ұжҳҜз”ҹжҙ»жң¬иә«зҡ„ж•°жҚ®еҢ–е‘ҲзҺ°гҖӮ

### еӣӣеӨ§жҳ е°„еҺҹеҲҷ

| зҺ°е®һжҰӮеҝө | жёёжҲҸжҰӮеҝө | и®ҫи®Ўеҗ«д№ү |
|---------|---------|---------|
| **ж—¶й—ҙ** | **иҙ§еёҒ** | ж—¶й—ҙжҳҜе”ҜдёҖдёҚеҸҜеҶҚз”ҹзҡ„иө„жәҗгҖӮиҠұеңЁе“ӘйҮҢпјҢе°ұжҠ•иө„еңЁе“ӘйҮҢгҖӮжөӘиҙ№ж—¶й—ҙ = иҙ§еёҒиҙ¬еҖјгҖӮ |
| **жҠҖиғҪ** | **иө„жң¬** | жҠҖиғҪеҸӘиғҪйҖҡиҝҮж—¶й—ҙжҠ•е…ҘжқҘеўһй•ҝпјҢжҠҖиғҪи§Јй”Ғж–°зҡ„еҸҜиғҪжҖ§пјҲд»»еҠЎгҖҒиЈ…еӨҮгҖҒжҲҗе°ұеүҚзҪ®жқЎд»¶пјүгҖӮ |
| **жҲҗе°ұ** | **йҮҢзЁӢзў‘** | жҲҗе°ұдёҚжҳҜ"е®ҢжҲҗдәҶеӨҡе°‘д»»еҠЎ"пјҢиҖҢжҳҜ"дҪ зҡ„дәәз”ҹеҲ°иҫҫдәҶд»Җд№Ҳй«ҳеәҰ"гҖӮдёҚеҸҜйҖҶгҖҒж°ёд№…и®°еҪ•гҖӮ |
| **иЈ…еӨҮ** | **иғҪеҠӣеўһе№…** | иЈ…еӨҮдёҚжҳҜиЈ…йҘ°е“ҒпјҢжҳҜиў«еҠЁеўһзӣҠгҖӮ"ж—©иө·д»ӘејҸ"иЈ…еӨҮ = жҜҸж—Ҙз»ҸйӘҢ+20%гҖӮиЈ…еӨҮжңүиҖҗд№…еәҰпјҢйңҖиҰҒз»ҙжҠӨгҖӮ |

### и®ҫи®ЎзәўзәҝпјҲз»қдёҚеҸҜиҝқеҸҚпјү

1. **ж—¶й—ҙеҝ…йЎ»й“ҫжҺҘеҲ°жҠҖиғҪ** вҖ” и®Ўж—¶еҷЁеҒңжӯўж—¶пјҢж—¶й—ҙеҝ…йЎ»иҪ¬еҢ–дёәжҹҗдёӘжҠҖиғҪзҡ„з»ҸйӘҢеҖјгҖӮдёҚе…Ғи®ё"ж— ж„Ҹд№үи®Ўж—¶"гҖӮ
2. **жҠҖиғҪеҸӘиғҪйҖҡиҝҮж—¶й—ҙжҠ•е…Ҙеўһй•ҝ** вҖ” жІЎжңүжҚ·еҫ„гҖҒжІЎжңүиҙӯд№°еҚҮзә§гҖӮ1е°Ҹж—¶ = еҜ№еә”жҠҖиғҪзҡ„з»ҸйӘҢеҖјгҖӮ
3. **иЈ…еӨҮеҝ…йЎ»жңүе®һйҷ…ж•Ҳжһң** вҖ” жІЎжңүзәҜиЈ…йҘ°жҖ§иЈ…еӨҮгҖӮжҜҸд»¶иЈ…еӨҮйғҪжңүеҸҜи®Ўз®—зҡ„иў«еҠЁж•ҲжһңгҖӮ
4. **ж¶ҲиҖ—жҖ§жҙ»еҠЁеҝ…йЎ»еҸҜи§Ғ** вҖ” еҲ·жүӢжңәгҖҒж— зӣ®зҡ„жөҸи§Ҳзӯү"ж¶ҲиҖ—жҖ§ж—¶й—ҙ"еҝ…йЎ»иў«и®°еҪ•пјҢдёҺ"жҠ•иө„жҖ§ж—¶й—ҙ"еҪўжҲҗеҜ№жҜ”гҖӮ
5. **UI еҝ…йЎ»жҳҜжёёжҲҸз•Ңйқў** вҖ” дёҚжҳҜ"еёҰеғҸзҙ йЈҺзҡ„еҫ…еҠһдәӢйЎ№ App"гҖӮжҳҜ"RPG жёёжҲҸз•ҢйқўпјҢж•°жҚ®жқҘиҮӘзңҹе®һз”ҹжҙ»"гҖӮ

## 2. еҪ“еүҚзі»з»ҹзҡ„й—®йўҳиҜҠж–ӯ

### 2.1 ж №жң¬й—®йўҳпјҡRPG жҳҜ"зҡ®"дёҚжҳҜ"йӘЁ"

еҪ“еүҚзі»з»ҹзҡ„жң¬иҙЁжҳҜдёҖдёӘ**дј з»ҹеҫ…еҠһдәӢйЎ№еә”з”Ё**пјҢдёҠйқўиҙҙдәҶ RPG иҙҙзәёпјҡ

- **д»»еҠЎзі»з»ҹ** (`TaskEntity`) жң¬иҙЁжҳҜ todo listпјҡtitleгҖҒstatusгҖҒdeadlineгҖҒpriorityгҖӮе’Ңд»»дҪ• todo app дёҖж ·гҖӮ
- **е®ҢжҲҗд»»еҠЎзҡ„еҘ–еҠұ** жҳҜзЎ¬зј–з Ғзҡ„пјҡдё»зәҝ=100exp, ж”Ҝзәҝ=50exp, ж—Ҙеёё=20expгҖӮжІЎжңүе’Ңе®һйҷ…жҠ•е…Ҙзҡ„ж—¶й—ҙе…іиҒ”гҖӮ
- **и®Ўж—¶еҷЁ** (`TimerViewModel`) е’Ң**жҠҖиғҪ** (`SkillEntity`) е®Ңе…Ёж–ӯејҖ вҖ” и®Ўж—¶еҷЁжңү categoryпјҲwork/study/sport...пјүпјҢжҠҖиғҪд№ҹжңү categoryпјҲprofessional/language/sport...пјүпјҢдҪҶдёӨиҖ…жІЎжңүжҳ е°„е…ізі»гҖӮ
- **е•Ҷеә—** (`ShopItemEntity`) зҡ„зү©е“ҒжҳҜзәҜиЈ…йҘ°пјҡеӨҙеғҸгҖҒз§°еҸ·гҖҒдё»йўҳй…ҚиүІгҖӮeffectType еҸӘжңү `avatar_style`гҖҒ`title_prefix`гҖҒ`theme_color`пјҢжІЎжңүе®һйҷ…жёёжҲҸж•ҲжһңгҖӮ
- **жҲҗе°ұ** (`AchievementEntity`) еҹәдәҺз®ҖеҚ•и®Ўж•°пјҲе®ҢжҲҗд»»еҠЎж•°гҖҒжҠҖиғҪж•°пјүпјҢдёҚжҳҜдәәз”ҹйҮҢзЁӢзў‘гҖӮ

### 2.2 е…·дҪ“д»Јз Ғй—®йўҳ

| ж–Үд»¶ | й—®йўҳ |
|------|------|
| `TaskViewModel.completeTask()` | зЎ¬зј–з Ғ exp еҘ–еҠұпјҢдёҚе…іиҒ”ж—¶й—ҙжҠ•е…Ҙ |

| `SkillViewModel.addSkillExp()` | жүӢеҠЁи°ғз”ЁпјҢжІЎжңүиҮӘеҠЁд»Һж—¶й—ҙжҠ•е…Ҙи§ҰеҸ‘ |
| `ShopRepository.seedDefaultItems()` | е…ЁжҳҜиЈ…йҘ°е“ҒпјҲеӨҙеғҸ/з§°еҸ·/дё»йўҳпјүпјҢж— иў«еҠЁж•Ҳжһң |
| `AchievementManager` | еҸӘжЈҖжҹҘз®ҖеҚ•и®Ўж•°жқЎд»¶пјҢдёҚжҳҜйҮҢзЁӢзў‘ејҸжҲҗе°ұ |
| `CharacterRepository.addExp()` | з»ҸйӘҢжқҘжәҗдёҚйҖҸжҳҺпјҢдёҚзҹҘйҒ“дёәд»Җд№ҲиҺ·еҫ—з»ҸйӘҢ |
| `NavGraph` / `Screen.kt` | 10дёӘеұҸе№•дҪҶж ёеҝғеҫӘзҺҜж–ӯиЈӮпјҢз”ЁжҲ·еңЁ"д»»еҠЎ"е’Ң"и®Ўж—¶"д№Ӣй—ҙи·іиҪ¬жІЎжңүе…іиҒ” |
